## 核心概念

玩家通过九关递增难度的博弈试炼，每关由 AI 天神镇守，博弈形式完全不同。从简单的信号博弈到复杂的元博弈，通关率仅 ~5%。

## 技术栈

- 前端：Web / 微信小程序
- 后端：Python + FastAPI
- AI：LLM API（Claude/GPT）
- 数据：SQLite
